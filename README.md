# mern-day-19-student-signup-login-dashboard
Student Signup Login Dashboard
A simple React.js application that allows students to register, log in, and access their dashboard.

Features
- Student Signup Form
- Student Login Authentication
- Dashboard Display
- Form Validation
- State Management using React Hooks
- Conditional Rendering
- Responsive User Interface
  
Technologies Used
- React.js
- JavaScript
- CSS3
- Vite

Concepts Practiced
- useState Hook
- Form Handling
- Event Handling
- Conditional Rendering
- Authentication Logic
- Component-Based Architecture
  
Learning Outcome

This project helped me understand how user authentication flow works in React applications, including signup, login, dashboard navigation, and state management.
